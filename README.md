# 🚀 Task Manager Full Stack Application

A modern full-stack Task Management application built using **Spring Boot** and **JavaScript**, featuring a clean UI, dynamic interactions, and real-world backend architecture.

---

## ✨ Features

- ✅ Create, update, delete tasks (CRUD)
- 🔄 Partial updates (null-safe handling)
- 🔍 Search tasks by title
- 🎯 Filter by status and category
- 🎨 Modern UI with animations & transitions
- 🔔 Toast notifications for actions
- ⚡ Real-time UI updates using Fetch API

---

## 🏗️ Project Structure
task-manager-fullstack/

## 🧠 Key Concepts Implemented

- DTO & Mapper pattern
- Layered architecture (Controller → Service → Repository)
- Exception handling (Global Handler)
- Partial updates using null checks
- REST API integration with frontend
